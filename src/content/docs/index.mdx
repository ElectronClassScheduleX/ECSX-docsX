## 使用说明

- 建议先阅读 [快速入门](/ecs/quickstart/) 完成安装.
- 完成安装后按顺序阅读 [课表配置教程](/ecs/setschedule/) 与 [功能介绍](/ecs/functionintroduction/).

## 学习路径建议

1. **10 分钟启动**：完成安装并确认软件可以正常打开.
2. **30 分钟配置**：按教程配置课表与基础参数, 形成可用环境.
3. **按需深入**：根据实际需求重点阅读功能介绍与更新说明.
